# AsonisCalculator
Java Calculator

A basic calculator project built using Java programming language with normal arithmetic functions such as addition, subtraction, multiplication, and division.
Requirements

    Java SE Development Kit 8 or higher
    Any Java Integrated Development Environment (IDE)

Features

    Basic arithmetic operations: addition, subtraction, multiplication, and division
    User-friendly interface

Classes

The following classes make up the Java Calculator project:

    Main - The entry point of the program. Contains the main method that runs the interface.
    Calculator - The core of the calculator, containing the logic for performing arithmetic operations.
    Operations - Enum class containing the list of supported operations.

Usage

    Clone the repository to your local machine
    Open the project in your IDE
    Run the Main class
    Follow the prompt to perform a calculation
